Introduction

Hello there world!

This is Display! A less than 2KB HTML presenter with easy CSS customization!

Display provides a way to show HTML code in a presentation format.

While it is possible (and very easy) to add markdown support / conversion if needed, it also takes up more space.


You can use the arrow < > keys to navigate between slides

text-align:h2:center background-color:darksalmon font-family:p:helvetica font-family:ul:helvetica

CSS (1/2)

CSS can be accessed directly from HTML, and be different on each slide.

This is what the CSS from the HTML document looks like for this slide:

text-align:h2:center
background-color:darksalmon
font-family:p:helvetica
fontFamily:ul:helvetica

As you can see on this slide, the title at the top is centered as shown in the CSS

JavaScript by default allows you to access styles with camelCase as well. So you could write the CSS above like so:

textAlign:h2:center
backgroundColor:darksalmon
fontFamily:p:helvetica
fontFamily:ul:helvetica
text-align:h2:center background-color:darksalmon font-family:p:helvetica font-family:ul:helvetica

CSS (2/2)

It is pretty straightforward:

a:b:c
  • a: is the css property you want to change for the ELEMENT
  • b: is the tag you want to affect
  • c: is the value the CSS property should have

So the following would select all <p> tags and give it the color blue: 

color:p:blue

When there's only:

a:b
  • a: is the CSS property you want to change for the SLIDE
  • b: is the value the CSS property should have

So to make all the text on the slide bold:

fontWeight:bold
